Взять немножечко укропу,
Сунь туда кошачью попу,
Тараканьи лапки,
Сикель старой бабки,
Один носок,
Пизды кусок,
Ведро воды
И хуй туды,
Два подпиздника телячьих,
Бычий хуй
И три собачьих.
На медленном огне варить,
"К ебёной мати" говорить,
Снимать ложкой пенку,
Биться головой о стенку,
Охапка дров,
И плов готов!

Суп кандык
Кусок манды
Лавровый листик
Бабе дристик
Хуя кусок
Пизды кусок
Смешать до дна
Получится сметана одна.
